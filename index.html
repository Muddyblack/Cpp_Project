@subsection autotoc_md2 Startup

To start working on this project so everybody has a similar version follow this Guide:

@subsubsection autotoc_md3 Windows

- Download the 64-Bit Mingw from <a href="https://winlibs.com/" >here</a>
- Extract the <tt>mingw</tt> to the place you want it to be
- Add <tt>Path_to_mingw/mingw/bin</tt> to your environment variables
- Download and extract <a href="https://www.boost.org/users/download/" >Boost</a>
- <tt>cd path_to_your_boost_folder</tt>
- <tt>bootstrap.bat mingw</tt>
- <tt>./b2 toolset=gcc</tt> (this will take a looong time)
- (Optional) Copy the <tt>boost</tt> folder to <tt>mingw/include</tt> and the <tt>stage/lib</tt> folder to <tt>mingw</tt>
- Install VS-Code
- Add <tt>C/C++ Extension Pack</tt> Extension
- Add <tt>Cmake Tools</tt> Extension
- Use <tt>ctrl+shift+p</tt> and select <tt>CMake: Configure</tt> and after <tt>CMake: Build</tt>

@subsubsection autotoc_md4 Linux

- <tt>sudo apt update</tt>
- <tt>sudo apt install libboost-all-dev</tt>
- Install VS-Code
- Add <tt>C/C++ Extension Pack</tt> Extension
- Add <tt>Cmake Tools</tt> Extension
- Use <tt>ctrl+shift+p</tt> and select <tt>CMake: Configure</tt> and after <tt>CMake: Build</tt>
